# Portable Development System

Terminal-first, AI-assisted dev methodology using worktrees for isolation.

## Available Skills
- `/ethos` - Development principles (read when stuck)
- `/worktree`, `/wt` - Worktree workflow and commands
- `/commit` - Semantic commit format
- `/review` - Code review checklist
- `/debug` - Systematic debugging
- `/design` - Architecture decisions
- `/test` - Test strategy
- `/bootstrap` - New project setup
- `/quickref` - Command cheatsheet

## Key Commands
- `wty` / `wtyg` - Open worktree in tmux/Ghostty layout
- `wta -b feature/x` - Create new worktree + branch
- `clauder` - Resume Claude session for current directory
